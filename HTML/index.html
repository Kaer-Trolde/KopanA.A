Немного о курсе

Курс посвящен базовым технологиям веб-программирования – HTML и CSS. Цель курса – научить "с нуля" создавать современные веб-интерфейсы, работая с кодом вручную.Вы сможете самостоятельно создавать веб-страницы начального и среднего уровня сложности. Полученный уровень подготовки позволит вам начать карьеру HTML-верстальщика или другого интернет-специалиста и применять полученные знания при решении широкого спектра задач, с которыми сталкивается большинство программистов

    Основные разделы дисциплины:
  • Информационная сеть WWW
  • Язык гипертекстовой разметки HTML
  • Форматирование HTML-документов
  • Динамический HTML
  • CSS

В результате изучения дисциплины студент должен:
    Знать
  • Терминологию и базовые понятия web-технологий
  • Архитектуру служб WWW и ее составляющих
  • Структуры HTML-документа
  • Основные теги языка HTML
  • Основные принципы каскадных таблиц стилей
    Уметь
  • Создавать web-приложения
  • Составлять HTML-документ средствами языка HTML
  • Применять каскадные таблицы стилей
  • Позиционировать компоненты HTML-страниц
  • Осуществлять поиск информации
    Владеть
  • технологиями и языками разработки web-приложений
